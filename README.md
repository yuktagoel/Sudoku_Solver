# Sudoku Solver

Sudoku is a popular number puzzle that requires you to fill blanks in a 9X9 grid with digits so that each column, each row, and each of the nine 3Ã—3 subgrids contains all of the digits from 1 to 9. In this project, I have implemented Backtracking Algorithm to solve the Sudoku.

- The web app is built on React.js with Material-UI.

- Wanna play again? Just click the navbar!
